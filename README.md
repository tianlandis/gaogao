# 2026 山东音乐类志愿填报助手 🎵

> 输入文化分 + 专业分，一键生成冲/稳/保梯度志愿推荐，支持 Excel/CSV 导出。

---

## 快速开始

```bash
# 安装 Python 依赖（仅首次）
pip install pandas openpyxl xlrd

# 生成数据（仅数据源更新时需要）
python build_real_data.py

# 启动服务
node serve.js
```

浏览器访问 **http://localhost:8080**

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 等位分换算 | 用位次桥接2026→2025分数，公平比较投档线 |
| 冲/稳/保梯度 | 基于等位综合分与投档线差值自动分级 |
| 多维筛选 | 省份、性质、层次、批次自由组合 |
| 多格式导出 | Excel / CSV / 复制志愿代码到剪贴板 |

---

## 算法简述

```
输入：文化分 + 专业分 + 方向（声乐/器乐/音教）

STEP 1  原始综合分 = 文化×0.5 + 专业×2.5×0.5
STEP 2  查2026专业位次（一分一段表）
STEP 3  位次反推2025等位专业分
STEP 4  等位综合分 = 文化×0.5 + 等位专业分×2.5×0.5

梯度：冲(diff≥-5) / 稳(diff≥0) / 保(diff>15)
```
